PredLactamase

PredLactamase is a Chou's pseudo amino acid composition and support vector machine based two level predictor which predict Betalactamase and its classes. At first level the predictor decided whether the input sequence(s) are from Beta-lactamase or non-Betalactamase on the basis of SVM score and threshold value. If the SVM score are greater than threshold, predictor predict the query sequence as betalactamase and move second level, else the programme will stop automatically. At second level the predictor clasify it one of the following four classes.

  1. Class A
  2. Class B
  3. Class C
  4. Class D

We found the overall prediction accuracy of PredLactamase at first level is 90.63% and at second level the maximum accuracy are 61.82%, 89.09%, 70.91% and 70.91% of class A, class B, class C and class D respectively.
