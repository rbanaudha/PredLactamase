How to Use

Query Sequence(s)
PredLactamase process only protein sequence file (fasta file). User have option to cut and paste in the text box or directly upload the sequence file. The limits of PredLactamase is upto 25 sequences at a time. If user submit more than 25 sequences, PredLactamase predict only first 25 sequences. Following figure 1 is an example showing how to submit a query in PredLactamase.

SVM Threshold Value
User also have the option to set the threshold. If user select high threshold, the number of prediction will be low but rate of false positive prediction will also be low. If the user select low threshold, the number of prediction will be high and the false positive prediction will also be high.
